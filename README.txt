ДЗ - 8

В файле MainTest.java предусмотрена демонстрация работы созданных классов, согласно функционалу, описанному исходными
условиями к ДЗ.

Также, для упрощения вывода в консоль, добавлены метод toArray и переопределен метод toString.

Для простоты структурирования были сформированы интерфейсы.

Для реализации алгоритмов работы классов MyQueue и MyStack, по условию задачи, было выполнено с помощью Node,
как и для MyLinkedList. По этой причине можно было бы также реализацию некоторых (одинаковых) методов этих классов
вынести в отдельный, промежуточный абстрактный класс, но за недостатком времени пришлось ограничиться такой реализацией ((
Главное, выполняются поставленные условия задачи)))
